# max file descriptors

## /etc/security/limits.conf

- 添加如下配置

```shell
* soft nofile 655360
* hard nofile 655360
* soft nproc 655360
* hard nproc 655360
```

## /etc/sysctl.conf (root)

- 添加如下配置

```shell
vm.max_map_count=655360
```

- 执行

```bash
sysctl -p
```
